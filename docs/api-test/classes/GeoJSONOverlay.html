    Class GeoJSONOverlay

    Overlay for rendering GeoJSON data.

    Renders GeoJSON geometries (points, lines, polygons) as an overlay with customizable styling.

    Example

    const overlay = new GeoJSONOverlay({
      geojson: myGeoJSON,
      fillStyle: 'rgba(255, 0, 0, 0.5)',
      strokeStyle: '#ff0000',
      strokeWidth: 2,
      pointRadius: 5
    });
    plugin.addOverlay(overlay);
