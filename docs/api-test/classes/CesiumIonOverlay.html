    Class CesiumIonOverlay

    Overlay for Cesium Ion imagery assets.

    Supports various external tile sources through Cesium Ion including Google Maps and Bing Maps.

    Example

    const overlay = new CesiumIonOverlay({
      assetId: 12345,
      apiToken: 'your-cesium-ion-api-token'
    });
    plugin.addOverlay(overlay);
